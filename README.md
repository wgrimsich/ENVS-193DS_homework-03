# ENVS-193DS_homework-03

# General Information

This repository contains statistical analyses and data visualizations for UCSB's ENVS193 DS course Homework 3 assignment. The analyses explore correlation strengths between ocean temperatures and giant kelp frond elongation rate, exploration in changes of personal screen time across different types of exercise and a continued critique of a chosen paper's data visualization efficacy.

# Data & File Information

"here" package utilized for reading in data with existing root directory.

## Folders

"data" consists two .csv files:
- "exercise.csv" consists of personal data regarding screen time, exercise time/type and additional variables.
- "temp-kelp.csv" consists of ocean temperature (ºC) and giant kelp frond elongation rate (cm/day) data.

"code" consists of three .pdf files, one .qmd markdown file, and one .jpeg image.
- "Exercise_Draft.pdf" consists of an artistic interpretation draft work of Figure 2.
- "Exercise_Type.pdf" consists of a rough draft sketch of Exercise_Draft.
- "Homework-03.pdf" consists of the entirety of this assignment output as a PDF and linked under 'Rendered Output' below.
- "Homework-03.qmd" consists of the markdown file encoding the Homework-03 PDF output.
- "figure-7.jpeg" consists of the graph being aesthetically critiqued in Problem 4.
